# Sandoval Commercial Disposal

Lead-generation website for **Sandoval Commercial Disposal** – junk removal, hauling, and demolition services in Southern California.

## Tech Stack

- Next.js 16, React 19, TypeScript
- Tailwind CSS 4
- App Router

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Configuration

- **Business info:** Edit `src/lib/constants.ts` (phone, email, service area)
- **Images:** Place in `public/`

## Deploy

Deploy to [Vercel](https://vercel.com) – connect this repo and it will auto-build.
